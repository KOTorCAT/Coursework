
#   менеджер задач в Docker

Веб-приложение для управления задачами. Работает в Docker.

## О проекте

Это учебный проект для курсовой работы. Приложение позволяет создавать задачи, назначать им приоритет и срок выполнения, отмечать выполнение и фильтровать список. Вся архитектура контейнеризирована и запускается одной командой.

Проект состоит из двух контейнеров:
- **Backend** — API на Python/Flask, работает с базой SQLite
- **Frontend** — статика на Nginx, проксирует запросы к API

Контейнеры связаны через внутреннюю Docker-сеть. Данные базы хранятся в Docker-томе.

## Что внутри

- Бэкенд — Python + Flask
- Фронтенд — HTML/CSS/JS
- База данных — SQLite
- Веб-сервер — Nginx

## Как запустить

```bash
docker compose up -d --build
```

Открыть в браузере: **http://localhost:8080**

## Как остановить

```bash
docker compose down
```

## Как удалить всё (включая базу)

```bash
docker compose down -v
```

## Что умеет

- Добавлять задачи
- Отмечать выполненные
- Устанавливать приоритет (низкий/средний/высокий)
- Указывать срок выполнения
- Фильтровать задачи (все / активные / выполненные)

## API

| Метод | Адрес | Что делает |
|-------|-------|-------------|
| GET | /api/tasks | Получить все задачи |
| POST | /api/tasks | Добавить задачу |
| PUT | /api/tasks/1 | Изменить статус |
| DELETE | /api/tasks/1 | Удалить задачу |

## Технологии

- Python 3.11
- Flask
- Nginx
- Docker
- SQLite
